Maintenance

Operational tasks that used to need a CLI session. Everything below is safe to run from the browser.

Resync Telegram dialogs

Re-resolve every group's name + profile photo against Telegram. Use when an "Unknown chat" lingers in the list.

Restart monitor

Stops and re-starts the realtime monitor in-process. Picks up settings / proxy / account changes without bouncing the container.

Database integrity check

Runs PRAGMA integrity_check against the SQLite catalogue. Read-only.

VACUUM database

Reclaims disk space after lots of deletions. Briefly locks the DB — avoid during a heavy backfill.

Verify files on disk

Walk every catalogued download, drop the rows whose file went missing on disk. The same sweep runs hourly in the background — this button forces it now.

Find duplicate files

Computes SHA-256 for every file and groups byte-identical copies. First scan is slow on large libraries; subsequent scans only hash newly-downloaded files. You review the duplicates and pick which copies to keep before anything is deleted.

Active share links

View, search, and revoke every shareable media link issued from this dashboard. Share links let a friend stream or download a file without logging in.

Build thumbnails for older files

Generates compact WebP previews for every file in the library. New downloads are thumbnailed automatically; this sweep covers anything that landed before the feature shipped.

Install update

Pulls the latest container image and recreates the container in place. The data volume and config are preserved; the SQLite database is snapshotted to data/backups/ first. The dashboard reconnects automatically once the new container is healthy.

Scan images for NSFW (18+)

Scans every photo in the library and flags ones the classifier thinks are NOT 18+ — likely false matches that need to be purged. Runs locally (no upload). The model downloads once on first use (~80 MB).

Download log file

Grab the tail of any logfile under data/logs/ as a plain-text download. Useful when filing bug reports.

View raw config.json

Open the on-disk config (with secrets redacted) in a viewer sheet. For audit / sanity-check.

Export Telegram session

Decrypts a saved data/sessions/<id>.enc and shows the gramJS string so you can migrate the account to another instance. Sensitive — anyone with this string can act as the account.

Sign out everywhere

Revokes every dashboard session token. You'll be sent back to the login page. Use this after rotating the password or if you suspect a compromise.
